# Configuração Linux

- Ver o IP
```
show
```

- IP 
```
ifconfig eth0 10.0.0.1 netmask 255.255.0.0 up
```

- Gateway
```
route add default gw 10.0.0.1 eth0
```

- DNS
```
/etc/resolv.conf
nameserver 200.192.168.1 
```

- Teclado
```
loadkeys br-abnt2
```

- Verificação do HD
```
df -h 
```

### Verificar a versão 
```
dpkg -l eve-ng
```

### Atualização

``` 
  apt autoremove
  apt update
  apt install eve-ng
  apt upgrade
  apt dist-upgrade
  ( reboot if needed  - see homepage instructions )
```
