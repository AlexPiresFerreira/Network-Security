### VMWare

- Fazer backup da VM para editar
```
Snapshot
``` 

- Processors
```
> Editar a VM
	> Processors
		> Marcar o opção
			--> Virtualize Intel VT-x/EPT or AMD-V/RVI
```

- Criar seguimento de rede

``` 
Edit
	Virtual Network Editor
		Add Network
		   11 até 19
			Desmarca
				Connect a host virtual adapter to this network
				User local DHCP server to distribute IP address to VMs
		   apply        
```

- Associar o seguimento de rede a VM

```
	Virtual Machine Settings
		add 
			network
				Custom: Specific virtual network
					1 até 9
```

	- Cloud 1 - VMnet1
	- Cloud 2 - VMnet2
	- Cloud 3 - VMnet3
	- Cloud 4 - VMnet4

### Configuração Linux

- Ver o IP
```
show
```

- IP 
```
ifconfig eth0 10.0.0.1 netmask 255.255.0.0 up
```

- Gateway
```
route add default gw 10.0.0.1 eth0
```

- DNS
```
/etc/resolv.conf
nameserver 200.192.168.1 
```

- Teclado
```
loadkeys br-abnt2
```

- Verificação do HD
```
df -h 
```
