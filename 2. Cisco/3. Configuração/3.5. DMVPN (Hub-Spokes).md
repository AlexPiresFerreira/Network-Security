# DMVPN (Hub-Spokes)
</br>

### Hub
</br>

```
Configuração do tunnel
R1(config)#interface tunnel <N>
R1(config-if)#ip address <IP_Mask>
 
Habilitar o DMVPN
R1(config-if)#ip nhrp map multicats dynamic
 
Configurar a ID do DMVPN todos os roteadores tem que ter o mesmo ID para participar do grupo 
R1(config-if)#ip nhrp network-id <Nº_da_ID>
 
Definir o endereço de origem
R1(config-if)#tunnel source <interface_de_origem>
 
Habiliatr o protocolo GRE no tunnel para o DMVPN para mais de uma filial
R1(config-if)#tunnel mode gre multipoint
 
Desabilitar o split-horizon no eigrp
R1(config-if)#no ip split-horizon eigrp <AS>
 
Tira o HUB como o próximo salto na filiais para ter conexão direta entre a filiais sem passar pela matriz
R1(config-if)#no ip next-hop-self <protocolo> <AS_Processo>   
```
</br>

### Spokes
</br>

```
Configuração do tunnel
R1(config)#interface tunnel <N>
R1(config-if)#ip address <IP_Mask>
 
Habilitar o DMVPN
R1(config-if)#ip nhrp map multicats <IP_da_matris_q_ira_gerenciar_o_DMVPN>
R1(config-if)#ip nhrp map <IP_do_tunnel_matris> <IP_da_matris>
 
Apontar o nhrp dentro do tunnel
R1(config-if)#ip nhrp <IP_do_tunnel_matris> 
 
Configurar a ID do DMVPN todos o roteadores tem que ter o mesmo ID 
R1(config-if)#ip nhrp network-id <Nº_da_ID>
 
 
Definir o endereço de origem
R1(config-if)#tunnel source <interface_de_origem>
 
Habiliatr o protocolo GRE no tunnel para o DMVPN para mais de uma filial
R1(config-if)#tunnel mode gre multipoint
```
</br>
