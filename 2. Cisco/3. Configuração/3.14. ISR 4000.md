```
!

bridge 1 route ip 
!

 no ip address
 load-interval 30

  encapsulation untagged
  bridge-domain 1
no shut
!
interface GigabitEthernet0/0/2
 description *** LAN DO CLIENTE ***
 no ip address

  encapsulation untagged
  bridge-domain 1
no shut
! 
!
```

### IP FLOW

```
!
flow exporter <Nome_1>
destination <IP de destino>
source <Interface de oriogem>
transport udp <Porta servidor>
export-protocol netflow-v5
!
flow monitor <Nome_2>
 exporter <Nome_1>
 cache timeout active 60
 record netflow-original
!
interface GigabitEthernet0/0/0
 ip flow monitor <Nome_2> output 
 
flow record TCP-APP-analysis 
 match ipv4 source address 
 match ipv4 destination address 
 match transport tcp source-port 
 match transport tcp destination-port 
 collect counter bytes 
 collect counter packets 
```

```
show flow monitor <Nome_2> cache format table 
```

### CAR - RATE-LIMIT: 

```
! Valor de BC é em bytes (SPO/IP/47349)
! Valor de PIR é em bits (Calcular PIR = BC*2 * 8)
 
class-map match-all <Nome_class-map>
match access-group <ACL>
 
policy-map <Nome_da_Policy>
class <Nome_class-map>
police cir <BW>000 bc <BW/>000 pir <BW/*>000
conform-action transmit
exceed-action drop
 
policy-map SaidaWAN
 class BestEffort
  bandwidth percent 35
service-policy <Nome_da_Policy>
 
access-list ACL permit ip host <Origem> <Destino>
 
interface GigabitEthernet0/0/0
 service-policy output SaidaWAN
```
