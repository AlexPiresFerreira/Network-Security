# SNMP  

### SNMPv3  

- Define User name, password, authentication method

```
!#### SNMP V3 READ no authentication ####
snmp-server group <group_name> v3 noauth
snmp-server user <user_name> <group_name> v3
snmp-server host 10.10.10.1 version 3 noauth <user_name>
```
</br>

```
!### SNMP V3 WRITE no authentication ###
snmp-server view <view_name> iso included
snmp-server view <view_name> system included
snmp-server group <group_name> v3 noauth write <view_name>
snmp-server user <user_name> <group_name> v3
snmp-server host 10.10.10.1 version 3 noauth <user_name>
```

</br>

```
!### SNMP V3 configuration with authentication ###
snmp-server group <group_name> v3 auth
snmp-server user <user_name> <group_name> v3 auth md5 <password>
snmp-server host 10.10.10.1 version 3 auth <user_name>
```

</br>

```
!### SNMP V3 configuration with authentication - encryption ###
snmp-server group <group_name> v3 auth
snmp-server user <user_name> <group_name> v3 auth md5 <password> priv aes 128 <password>
snmp-server host 10.10.10.1 version 3 auth <user_name>
```
