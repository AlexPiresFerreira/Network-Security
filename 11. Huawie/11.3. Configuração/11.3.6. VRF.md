# VRF

### Create and configure VPN 1

```
ip vpn-instance vpn1
 route-distinguisher 100:1
 vpn-target 100:1 export-extcommunity
 vpn-target 100:1 import-extcommunity
```
 
### Associate interfaces Ten-GigabitEthernet 0/0 with VPN 1.

```
interface Ten-GigabitEthernet0/0.101                                          
  ip binding vpn-instance vpn1
  ip address 10.1.1.1 255.255.255.0
```

### Configure a static route

```
ip route-static vpn-instance vpn1 0.0.0.0 0.0.0.0 <IP destination>
```
 
### ping

```
ping -vpn-instance vpn1 31.1.1.1
```
