### Habilitar o tracert 

```
ip unreachables enable 
ip ttl-expires enable
```

### Usuario e senha Read Only
 
```
role name rancid-role
 description Rancid role
 rule 1 permit command display *
 rule 2 permit command dir *
 rule 3 permit command undo terminal monitor
 rule 4 permit command screen-length disable
 quit
 
local-user rancid class manage
 service-type ssh
 authorization-attribute user-role rancid-role
```

### Quebrar a senha do router Huawei
 
3­ Quebra de senha em roteador huawei
 
• 1º Resetar o router e ir  precionando CTRL+b 
• 2º Caira no boot menu, escolher troca a opção “Start up and ignore configuration”(que nem sempre esta no numero 4 em alguns routers esta no 3).
 
• Boot Menu: (esta é a tela que ira aparecer) 
 
      1:  Download application program with XMODEM
      2:  Download application program with NET
      3:  Clear configuration
      4:  Start up and ignore configuration   
      5:  Boot ROM Operation Menu
      6:  Do not check the version of the software
      7:  Exit and reboot
      Enter your choice(1­7):4
 
• Obs: neste caso não é a opção 3 e sim a 4).
• 3º irá me perguntar se tenho certeza
• Start up and ignore configuration, are you sure?[Y/N]y (aperto y para confirmar )
  
4º Pedira usuario e senha, pressionar enter enter.
 
5º agora achar a linha que tiver a opção de reboot:
 
•   Boot Menu:    
      1:  Download application program with XMODEM
      2:  Download application program with NET
      3:  Clear configuration
      4:  Start up and ignore configuration
      5:  Boot ROM Operation Menu
      6:  Do not check the version of the software
      7:  Exit and reboot 
      Enter your choice(1­7):4
 
Obs: Esta na opção 7 mas em alguns routeadores esta em outro numero.
  
6º Importante: Para recuperar a configuração
• O arquivo onde estão as configurações salvas do cliente é o saved­configuration.
Comando:
  [Router]display saved­configuration (este é o arquivo de  inicialização como
 se fosse o startup­config do CISCO)
  
7º Se a senha não estiver criptografada é só ver que senha é, 
 
dar um “reboot”, quando pedir a opção salvar dar um “n” e 
ele subira com toda a configuração.
  
8º se a senha for criptografada
• Copiar o script para um editor de teste para depois coloca­lo outra vez 
no router, (muito importante ou ficara sem as configurações do cliente).
  
9º  Configurar usuário e senha no router:
     Comando:
• [teste]local­user silvia service­type administrator password simple silvia (configuração de de usuario com perfil de administrador e senha 
criptografada).
• Salvar para que fique só com a senha nova
• [Router]save
  
10º Reiniciar o router
• [Router]reboot
•   WARNING: Configuration has been changed !  Save configuration ? 
[Y/N]y (só dou um y para dizer que quero salvar isto só aparecera se 
eu não tiver salvo no passo 8).
•   Now writing the running config to flash memory.
•   Please wait for a while......
•   write the running config to flash memory successfully
•   WARNING: System will REBOOT!  Continue ?[Y/N]y (digito y 
para confirmar que quero rebootar)
  
11º Entrar com a nova senha, e colocar o script que foi  salvo no editor 
tirando fora só a parte de usuário e senha e salvar
