### Firewall
- Criar policy

```
set firewall name <name> default-action drop
set firewall name <name> description <description>
set firewall name <name> rule <N> description <description>
set firewall name <name> rule <N> action [accept| drop]
set firewall name <name> rule <N> state [established | related| invalid] enable
set firewall name <name> rule <N> protocol [tcp | udp | tcp_udp]
set firewall name <name> rule <N> destination address <IP>
set firewall name <name> rule <N> destination port <port>
set firewall name <name> rule <N> log [enable | disable]
```

- Aplicar policy

```
set interface ethernet <Interface> firewall [in | out | local] name <name_policy>
```

- Firewall/NAT group

```
set firewall group network-group <name> description <description>
set firewall group network-group <name> network <network/mask>
```

- DNAT 

```
set service nat rule <N> description <description>
set service nat rule <N> destination address <IP_WAN>
set service nat rule <N> destination port <port_WAN>
set service nat rule <N> inbound-interface <interface_WAN>
set service nat rule <N> inside-address address <IP_interno>
set service nat rule <N> inside-address port <port_interno>
set service nat rule <N> protocol [protocol]
set service nat rule <N> log [enable | disable]
set service nat rule <N> type destination
```

- Port-forward

```
set port-forward auto-firewall enable 
set port-forward hairpin-nat enable 
set port-forward wan-interface <interface_WAN>
set port-forward lan-interface <interface_LAN>
set port-forward rule <N> description <description>
set port-forward rule <N> forward-to address <IP_LAN>
set port-forward rule <N> forward-to port <port>
set port-forward rule <N> original-port <port>
set port-forward rule <N> protocol [tcp | udp]
```

- SNAT 

```
set service nat rule <N> description <description>
set service nat rule <N> outbound-interface <interface_WAN>
set service nat rule <N> type source
set service nat rule <N> protocol [all]
set service nat rule <N> outside-address address <IP_WAN>
set service nat rule <N> source address <IP_LAN>
```
