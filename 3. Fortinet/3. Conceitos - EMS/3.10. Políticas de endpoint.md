
### 10.1 Endpoint policy

- Define o que será aplicado ao grupo/usuário
- Pode incluir:
    - perfil do endpoint
    - regras de on-fabric detection
    - configurações por tipo de dispositivo

### 10.2 Endpoint profiles

Perfis citados:

- Remote Access
- ZTNA Destinations
- Web Filter
- Vulnerability Scan
- Malware Protection
- Sandbox
- Firewall
- System Settings

### 10.3 Prioridade

- Um endpoint pode ser elegível para várias políticas
- A prioridade define qual política será aplicada
