
### 5.1 Métodos de distribuição citados

- **SCCM**
- **GPO**
- **MDM**
- link de instalação enviado ao usuário
- pacote customizado do EMS

### 5.2 Pacotes de instalação

- MSI
- EXE
- ZIP
- DMG
- variação por arquitetura e sistema operacional

### 5.3 Pacotes de deploy

- Podem incluir:
    - versão do cliente
    - módulos
    - telemetria
    - VPN
    - web filter
    - sandbox
    - vulnerability scan
    - application firewall

### 5.4 Observação importante

- O EMS pode criar pacote de deployment, mas o pacote em si **não é editado depois**
- As mudanças são feitas via **perfil/política**
