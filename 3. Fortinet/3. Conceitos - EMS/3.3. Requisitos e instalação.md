
### 3.1 Requisitos mínimos citados

- Windows Server **2012 R2 ou superior**
- Processador **64-bit**
- CPU e RAM suficientes para a quantidade de endpoints
- Disco livre para base e logs
- Acesso à internet

### 3.2 Instalação via CLI

- O documento mostra instalação do EMS por parâmetros de linha de comando
- Isso permite definir:
    - porta de download
    - SQL
    - autenticação
    - diretório
    - gerenciamento remoto

### 3.3 Desinstalação

- Pode ser feita quando:
    - há migração para outro EMS
    - existe conflito com outro software
    - é necessária uma instalação limpa
