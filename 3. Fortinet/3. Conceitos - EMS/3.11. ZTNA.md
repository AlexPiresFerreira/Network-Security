
### 11.1 Conceito

- Acesso baseado em **identidade + postura + tags**
- Substitui ou complementa VPN tradicional
- Baseado em camada de aplicação, não em acesso amplo à rede

### 11.2 Componentes

- FortiClient
- FortiClient EMS
- FortiGate
- Access Proxy
- ZTNA destination
- Zero Trust tags

### 11.3 Tipos de acesso

- TCP forwarding
- SSH access
- access proxy
- service portal
- IP/MAC-based access control
- CASB inline

### 11.4 Tagging

- Regras que classificam endpoints dinamicamente
- Tags são sincronizadas do EMS para o FortiGate
