
### 15.1 Método sugerido

- levantar versão do FortiClient
- validar requisitos
- checar softwares conflitantes
- verificar mudanças recentes
- testar em outro endpoint
- analisar logs e debug

### 15.2 Ferramentas úteis

- FortiClient Diagnostic Tool
- logs exportados
- debug em EMS
- logs de endpoint
- tela de conexões e telemetry status

### 15.3 Problemas comuns

- endpoint não registra
- deploy falha
- upgrade falha
- telemetria não conecta
- certificado inválido
- policy não aplica
- quarentena indevida
