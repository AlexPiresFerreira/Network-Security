
### 8.1 Conta padrão

- Usuário padrão: **admin**
- Deve trocar a senha imediatamente

### 8.2 Perfis de administração

- Super administrator
- Standard administrator
- Endpoint administrator
- Restricted administrator

### 8.3 Multi-tenancy

- Permite separar:
    - clientes
    - sites
    - administradores
    - dados de endpoint
- Útil para:
    - MSSP
    - ambientes com múltiplas filiais
    - segregação por cliente
