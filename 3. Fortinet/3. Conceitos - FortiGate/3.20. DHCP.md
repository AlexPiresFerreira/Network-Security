
### 5.1 Requisitos

- Interface com:
    - **IP manual** (static)
    - **Role = LAN**

### 5.2 Configurações básicas

- Range de IP: ex. `192.168.10.2–192.168.10.254`
- Máscara, gateway, DNS
- Lease time (ex.: `604800` s = 7 dias)
- Opção de **criar address object** do subnet

### 5.3 Opções avançadas

- **Mode**:
    - Server / Relay / IPsec
- **NTP Server** (local/system/specify)
- **Wireless controller**
- **Time zone**
- **Next bootstrap server / TFTP**
- **Additional DHCP options**

### 5.4 Segurança (MAC / reserva)

- Por padrão: entrega IP para qualquer MAC
- Pode mudar para:
    - **block all** e liberar só MACs específicos
- Pode:
    - **reservar IP** por MAC
    - **bloquear MAC** específico
