
### Release notes do FortiOS

- Antes de fazer upgrade, consulte sempre as **release notes**
- Motivos:
    - verificar **known issues**
    - checar **caminho de upgrade**
    - confirmar compatibilidade
    - evitar problemas em produção

### Fluxo recomendado

1. Verificar a versão atual
2. Ler as **release notes**
3. Conferir o **upgrade path**
4. Validar impactos conhecidos
5. Só então executar o upgrade

### 11.2 Upgrade com licença ativa

- Recebe notificações de novas versões
- Pode:
    - fazer download direto da Fortinet via GUI
- Sem licença:
    - precisa baixar manualmente o firmware no site
    - fazer upload via GUI

### 11.3 Antes de atualizar

- Verificar:
    - **Upgrade path** (no suporte Fortinet)
    - **Release Notes** em `<docs.fortinet.com>`
    - **Known issues** na release note (checando impacto no ambiente)

### 11.4 Agendamento

- Em FortiGates com **disco**, é possível **agendar** upgrades
