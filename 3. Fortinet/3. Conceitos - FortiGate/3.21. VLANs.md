- Segmentam rede L2 em domínios de broadcast menores
- TAG VLAN é adicionada ao frame para identificar o segmento
- É possível ter múltiplas VLANs na mesma interface física (IDs únicos)

### 6.1 Criação de VLAN

- Definir:
    - tipo **VLAN**
    - interface física de origem
    - **nome**
    - **VLAN ID**
    - IP, DHCP server, role etc.

### 6.2 Identificação

- Tráfego **sem VLAN** → VLAN nativa (`VID 0`)
- Tráfego **com VLAN**:
    - frame recebe tag com **VLAN ID** → FortiGate encaminha para a subinterface correta

### 6.3 VLAN x Multi-VDOM

- Em **multi-VDOM**:
    - interface física pode estar em um VDOM
    - subinterface VLAN em outro
    - ex.: `port1` em **root**, `port1.100` em **PARCEIROS**
