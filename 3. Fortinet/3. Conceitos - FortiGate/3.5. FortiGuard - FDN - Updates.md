### 12.1 Função

- Serviço de inteligência de ameaças da Fortinet
- Usado para:
    - validar licença
    - baixar engines e assinaturas (AV, IPS, App Control etc.)
    - consultas em tempo real de **domínios/URLs** para:
    - Web Filter
    - DNS Filter
    - AntiSpam
    - uso de **DNS over TLS** (DoT) como DNS seguro

### 12.2 Requisitos

- Conexão à internet
- Contratos de assinatura específicos por serviço

### 12.3 URLs importantes

- `update.fortiguard.net` (TCP 443)  
    → updates de pacotes/assinaturas (AV, IPS etc.)
- `service.fortiguard.net` (UDP 53 ou 8888)  
    → consultas web filter / dns filter / antispam
- `securewf.fortiguard.net` (HTTPS 443, 53 ou 8888)  
    → consultas **seguras** (WEB/DNS/antispam)

### 12.4 FDN / anycast x multicast

- Conexão via **FortiGuard Distribution Network (FDN)**
- Data centers: Am. do Norte, Ásia, Europa
- Pode criar **FDN local** via FortiManager
- Modos:
    - **Anycast (default)**:
    - um único IP por domínio
    - consultas WF/DNS/antispam forçadas em HTTPS 443
    - **Multicast**:
    - FortiGate escolhe datacenter considerando time zone e carga
