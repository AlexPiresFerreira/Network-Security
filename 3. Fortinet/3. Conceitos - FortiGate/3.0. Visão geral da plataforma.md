## 2) Modos de operação

- **Modo padrão**: **NAT mode**
    - Atua como **roteador L3**
    - Todas as interfaces usadas precisam ter IP (estático ou dinâmico)
- Outros modos:
    - **Transparent mode**
    - **Virtual Wire mode**

## Fortinet Enterprise Firewall Solution

- **Contexto geral**
    
    - FortiGate presente em:
    - **Branch** (filiais) – FortiGate série **40–90**
    - **Campus** – FortiGate série **100–900**
    - **Data Center** – FortiGate série **1000–7000**
    - **VM / Cloud / FWaaS** – **FortiGate VM**, **FortiCNF**, **FortiSASE**
    - Gestão centralizada via **FortiManager** (console única).
- **Funções principais do FortiOS**
    
    - **Firewall**, **VPN**, **NGFW**, **segmentation**, **SD‑WAN segura**, **ZTNA**, inspeção SSL, hyperscale, etc.
    - Integração com **FortiGuard Labs** (inteligência de ameaças AI).

## 2) Tipos de firewall FortiGate (tabela DEFW / ISFW / DCFW / NGFW)

- **DEFW (Data Center Firewall)**
    
    - Recursos: firewall, NGFW, segmentação, inspeção SSL por hardware, **baixa latência**, **hyperscale**.
    - Propósito: proteção de rede de **alta velocidade** (data center).
    - Modelos: FortiGate **série 1000–7000**.
    - Throughput médio: **10–1000 Gbps**.
- **ISFW (Internal Segmentation Firewall)**
    
    - Recursos: firewall, NGFW, segmentação, baixa latência, **LAN sem fio integrada**, inspeção SSL por hardware.
    - Propósito: **microsegmentação** de rede (campus / core interno).
    - Modelos: FortiGate **série 100–900**.
    - Throughput médio: **1–100 Gbps**.
- **NGFW (Next‑Generation Firewall)**
    
    - Recursos: firewall, VPN, **D‑NGFW**, segmentação, proteção contra DDoS, baixa latência, LAN sem fio, 5G/LTE, inspeção SSL por hardware, **Secure SD‑WAN**, hyperscale, ZTNA.
    - Propósito: **segurança multicamadas** em diversos cenários (campus, data center, cloud).
    - Modelos: FortiGate **série 100–7000**.
    - Throughput médio: **1–40 Gbps**.
- **DEFW (Branch / Filiais)**
    
    - Recursos: firewall, VPN, segmentação, inspeção SSL por hardware, LAN sem fio integrada.
    - Propósito: proteção de **sites remotos** / filiais.
    - Modelos: FortiGate **série 40–90**.
    - Throughput médio: **até 1 Gbps**.

O **FortiGate** é o firewall de próxima geração da Fortinet, projetado para proteger redes corporativas com recursos de:
- inspeção de tráfego
- controle de aplicações
- VPN
- autenticação de usuários
- alta disponibilidade
- segmentação de rede
- prevenção contra ameaças

Ele pode atuar como firewall, roteador, gateway de VPN, proxy e plataforma de segurança unificada.

## Transparent Mode
No **Transparent Mode**, o FortiGate atua como uma **ponte/bridge**, sem necessidade de mudar o gateway da rede.

### Quando usar
- Quando não se quer alterar o endereçamento da rede
- Quando o firewall precisa ser inserido “no meio” do caminho
- Em cenários onde a mudança de arquitetura deve ser mínima

### Características
- Não funciona como roteador tradicional
- Inspeciona o tráfego mesmo sem atuar como salto de roteamento
- Permite proteção com impacto reduzido na estrutura existente

### Vantagens
- Implantação mais simples em alguns cenários
- Menos alterações na rede
- Útil em migrações e projetos de adaptação
