## 1) SPUs — unidades de processamento de segurança

### Principais módulos citados

- **CP9** — Content Processor
- **SP3** — Security Processor
- **NP6** — Network Processor
- **SoC4** — System-on-a-Chip

### Ideia geral

- **CP**: inspeção de conteúdo e funções de segurança mais pesadas
- **SP/NP**: aceleração de tráfego e inspeção em hardware
- **SoC**: otimização para equipamentos de entrada

## PPP — Parallel Path Processing

### 2.1 Conceito

- Processamento de tráfego em paralelo usando hardware dedicado
- Reduz carga do CPU principal

### 2.2 Processadores citados

|Tipo|Módulos|
|---|---|
|**Content Processor**|CP8, CP9|
|**Network Processor**|NP6, NP7|

### Tipos citados

- **CPs** → processadores de conteúdo
- **SPs** → processadores de segurança
- **NPs** → processadores de rede

### Ideia principal

- Esses blocos aceleram inspeção, roteamento e processamento de tráfego
- São a base da aceleração de hardware no FortiGate
