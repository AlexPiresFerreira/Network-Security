
O FortiGate precisa de acesso à Internet e DNS para resolver os serviços do FortiGuard e baixar atualizações de assinatura; os endpoints e portas mudam conforme o serviço usado  

### 2.1 Serviços e endpoints

- **Antivírus e IPS**
    
    - `update.fortiguard.net`
    - **TCP 443**  
        
- **Web Filter, DNS e Antispam**
    
    - `service.fortiguard.net`
    - **UDP 53** ou **UDP 8888**  
        
    - `securewf.fortiguard.net`
    - **HTTPS 443**, **HTTPS 53** ou **HTTPS 8888**  

### 2.2 Observação importante

- O FortiGate precisa conseguir resolver os FQDNs do FortiGuard via DNS para atualizar corretamente  
-
- Em troubleshooting de web filtering/antispam, `diagnose debug rating` é o comando principal para validar conectividade e qualidade da resposta do serviço
