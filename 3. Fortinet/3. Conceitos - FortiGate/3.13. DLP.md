File filtering is based only on the file type (file meta data) and not on file size or content. A DLP sensor would need to be configured to block files based on size or content, such as SSN numbers, credit card numbers, or regular expressions (see [Basic DLP filter types](https://docs.fortinet.com/document/fortigate/7.0.14/administration-guide/919417/basic-dlp-filter-types)for more information).
