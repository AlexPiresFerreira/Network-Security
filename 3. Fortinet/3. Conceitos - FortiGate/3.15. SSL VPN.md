A **SSL VPN** é usada para acesso remoto seguro.

### Finalidade
- Permitir acesso de usuários remotos aos recursos internos
- Criptografar a comunicação com SSL/TLS

### Cenários comuns
- Home office
- Acesso de consultores ou terceiros
- Conexão segura via navegador ou cliente dedicado

### Vantagens
- Fácil adoção para acesso remoto
- Boa usabilidade
- Segurança adequada para conexões externas

### Pontos de atenção
- Deve ser protegida com políticas fortes
- É importante controlar autenticação e permissões
