## 11) Resumo curto para revisão

- **Security Fabric**: diagnose via `csf upstream`, `csf downstream`, `csf neighbor list`
- **PPP**: aceleração por CP8/9 e NP6/7
- **Memória**: `diagnose hardware sysinfo memory`
- **CPU**: `diagnose sys top`
- **Conserve Mode**: thresholds via `system global`, verificação via `sysinfo conserve`
- **Sessões**: filtro → listar → limpar
- **Sniffer**: `diagnose sniffer packet`
- **Debug flow**: filtro → enable → trace start → trace stop
- **FIB**: `get router info kernel`
- **VRF**: até 32, sem suporte a RIP
- **Web Filter**: consulta em `fortiguard.com/webfilter`
