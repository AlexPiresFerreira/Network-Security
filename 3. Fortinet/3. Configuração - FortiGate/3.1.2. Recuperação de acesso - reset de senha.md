
### Acesso com usuário `maintainer`

Para acessar o FortiGate quando você não possui a senha de administrador:

1. Acesse o equipamento via **console**
2. Reinicie o equipamento
3. Quando o prompt de login aparecer, você terá cerca de **30 segundos** para informar:

```text
User: maintainer
Password: bcpb<SERIAL-NUMBER>
```

> **Importante:** o **serial number** deve ser digitado em **maiúsculas**.

---

### Alterar a senha de um administrador

Depois de entrar no equipamento, altere a senha do usuário desejado:

```bash
config system admin
    edit "diego"
        set password <nova-senha>
    next
end
```

---

### Desabilitar o acesso por `maintainer`

Se necessário, desative esse mecanismo por questão de risco ou compliance:

```bash
config system global
    set admin-maintainer disable
end
```
