
### Rota estática padrão

> **Observação:** substitua o gateway abaixo pelo **next-hop correto** da sua rede.

```bash
config router static
    edit 1
        set device "wan1"
        set gateway <gateway-da-rede>
    next
end
```

```
config firewall address

    next
```

### 11.3 FSSO

```bash
# Debug da comunicação Agent → FortiGate
diagnose debug application authd 8256

# Listar usuários autenticados via FSSO
diagnose debug authd fsso list

# Filtrar usuário específico
diagnose debug authd fsso filter user <user>

# Debug da comunicação com o AD
diagnose debug application fssod -1

# Usuários autenticados via FSSO Polling
diagnose debug fsso-polling user
```

> **Modos do Collector:**
> - **DC Agent Mode** → `dcagent.dll` em `%windir%\system32`
> - **Polling Mode** → NetAPI / Event Log / WMI

### 2.4 ECMP vs SD-WAN

- Quando **SD-WAN está habilitado**, o comportamento de ECMP muda.
- O documento compara:
    - ECMP tradicional
    - SD-WAN ECMP / load-balance-mode
- O SD-WAN pode usar:
    - peso do membro
    - spillover threshold
    - algoritmo de volume

#### Comando chave

```bash
get router info routing-table all
get router info kernel
diagnose firewall proute list
config system settings
    set strict-src-check enable
end
config system interface
    edit <interface>
        set src-check disable
    next
end
config system settings
    set v4-ecmp-mode source-ip-based
end
```
