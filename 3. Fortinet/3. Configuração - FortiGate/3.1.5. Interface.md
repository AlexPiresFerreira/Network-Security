###  Interface `internal`

```bash
config system interface
    edit "internal"
        set ip 192.168.1.254 255.255.255.0
        set allowaccess ping https http
        set type physical
    next
end
```

### Interface `wan1`

```bash
config system interface
    edit "wan1"
        set ip 111.222.111.2 255.255.255.248
        set allowaccess ping https
        set gwdetect enable
        set detectserver "111.222.111.123"
        set type physical
        set speed 10half
    next
end
```

### 4.2 Roles de interface

- **LAN**
- **WAN**
- **DMZ**
- **Undefined**

Impactos (exemplos):

- **DHCP**:
    - não aparece quando role é **WAN** ou **DMZ**
- **Device detection**:
    - não aparece quando role é **WAN**
- **Estimated bandwidth**:
    - só aparece quando role é **WAN**
- **Security Mode** (captive portal):
    - só aparece quando role é **LAN**
- Em roles **LAN/DMZ** pode aparecer opção para criar **address object** automaticamente

### 4.3 Alias

- Alias = “nome amigável” da interface
    - ex.: `LINK_ALGAR_500Mb`, `LAN`, etc.
- Aparece nas listas de policies
- Ajuda na leitura de logs e relatórios
