
## 1) Comandos e pontos-chave de CLI

- **Ver tabela do kernel / FIB**:

```bash
  get router info kernel
```

- **Habilitar detecção de aplicações no SD-WAN**:

```bash
  config system global
      set gui-app-detection-sdwan enable
  end
```

- **Verificar categorias de Internet Service / Application Control no SD-WAN**:

```bash
  diagnose sys sdwan internet-service-app-ctrl-category-list <id>
```

- **FortiManager scripts**:
    - **CLI** usa `$`
    - **Jinja2** usa `{{ }}`

```bash
# Consultar serviço por ID
diagnose internet-service id <id>

# Resumo do serviço por ID
diagnose internet-service id-summary <id>

# Informação detalhada (vdom, protocolo, serviço, IP)
diagnose internet-service info <vdom> <protocolo> <serviço> <IP>

# Match de IP com ISDB
diagnose internet-service match <vdom> <IP> <mask>

# Listar app-ctrl associados ao SD-WAN
diagnose sys sdwan internet-service-app-ctrl-list

# Verificar SD-WAN service por ID
diagnose sys sdwan service <id>
```
