### NAT

#### SNAT - Source NAT (IP Pool)

> Usado para traduzir o **endereço de origem** dos pacotes, geralmente para acesso à internet.

```bash
config firewall ippool
    edit "<nome-do-pool>"
        set startip <IP_inicial>
        set endip <IP_final>
    next
end
```

#### DNAT - Destination NAT (Virtual IP)

> Usado para expor serviços internos para o exterior, traduzindo o **endereço de destino**.

```bash
config firewall vip
    edit "<nome-do-vip>"
        set extip <IP_externo>
        set extintf <interface_externa>
        set mappedip <IP_interno>
        set portforward enable
        set extport <porta_externa>
        set mappedport <porta_interna>
    next
end
```

#### Política de Firewall com SNAT

> Use `edit 0` para que o FortiGate atribua automaticamente o próximo ID livre.

```bash
config firewall policy
    edit 0
        set name "<nome-da-regra>"
        set srcintf "<interface_origem>"
        set dstintf "<interface_destino>"
        set srcaddr "<endereco_origem>"
        set dstaddr "<endereco_destino>"
        set action accept
        set schedule "always"
        set service "<servico>"
        set nat enable
        set ippool enable
        set poolname "<nome-do-pool-SNAT>"
    next
end
```

### Criar serviço custom (ex: DNS)

```bash
config firewall service custom
    edit "DNS"
        set category "Network Services"
        set tcp-portrange 53
        set udp-portrange 53
    next
end
```

### Criar objeto de rede (subnet)

```bash
config firewall address
    edit "Mantos_Blancos_10.185.106.0/24"
        set subnet 10.185.106.0 255.255.255.0
    next
end
```
