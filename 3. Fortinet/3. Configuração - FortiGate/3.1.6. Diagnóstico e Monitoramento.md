### Informações do sistema

| Comando | Descrição |
|---|---|
| `get system performance status` | CPU, memória e uptime |
| `diagnose hardware sysinfo memory` | Detalhes da memória |
| `diagnose ip arp list` | Tabela ARP |
| `diagnose hardware deviceinfo nic <porta>` | Info da interface de rede |

```bash
# Informações da NIC (driver, MAC, duplex, velocidade)
diagnose hardware deviceinfo nic port<N>
get hardware nic port<N>

get hardware status

diagnose debug crashlog read

diagnose hardware deviceinfo disk

# Formatar disco de log
execute formatlogdisk

# Informações das interfaces físicas
get system interface physical
```
