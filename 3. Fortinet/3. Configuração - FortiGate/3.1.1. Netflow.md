
#### **Fortigate**

- Configuração do Netflow:

```
config system netflow
    set collector-ip <ip>                ---------> (IP DO WUG)
    set collector-port <port>            ---------> (A PORTA DO WUG(2055))
    set source-ip <ip>                   ---------> (IP DE ORIGEM)
    set active-flow-timeout <integer>    ---------> (60 - 3600, default = 1800)
    set inactive-flow-timeout <integer>  ---------> (10 - 600, default = 15)
    set template-tx-timeout <integer>    ---------> (60 - 86400, default = 1800)
    set template-tx-counter <integer>    ---------> (10 - 6000, default = 20)
end
```

- Configuração do Netflow em um VDOM específico:

```
config vdom
    edit <vdom>
        config system vdom-netflow
            set vdom-netflow enable
            set collector-ip <ip>         ---------> (IP DO WUG)
            set collector-port <port>     ---------> (A PORTA DO WUG(2055))
            set source-ip <ip>            ---------> (IP DE ORIGEM)
        end
    next
end
```

- Configuração do Sampler/Amostragem da interface:

```
config system interface
    edit <interface>
        set netflow-sampler {disable | tx | rx | both}
    next
end
```
