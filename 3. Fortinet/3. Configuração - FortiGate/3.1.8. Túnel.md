
#### 12.3 Túnel GRE

```bash
config system gre-tunnel
    edit "<name>"
        set interface "<port>"
        set local-gw <IP_local>
        set remote-gw <IP_remoto>
    next
end

config system interface
    edit "<name>"
        set vdom "vdom"
        set ip <IP> <mask>
        set type tunnel
        set remote-ip <IP_remoto>
        set interface <port>
    next
end
```
