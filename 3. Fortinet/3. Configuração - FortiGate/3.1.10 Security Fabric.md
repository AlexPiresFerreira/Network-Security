
### 2.3 Configuração

```bash
config system csf
    set configuration-sync local
    set fabric-object-unification local
end
```

- `configuration-sync local` → desativa sincronização upstream → downstream
- `fabric-object-unification local` → mantém objetos locais sem unificar com o fabric

> **Observação:** o comando abaixo indica que o objeto pertence ao fabric:


```bash
set fabric-object enable
```
