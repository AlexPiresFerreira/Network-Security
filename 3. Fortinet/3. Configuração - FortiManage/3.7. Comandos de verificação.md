## 17) Verificação da associação de dispositivos


```bash
diagnose debug reset
diagnose debug disable
diagnose debug application depmanager 0
diagnose debug application depmanager 255
diagnose debug enable

# Para encerrar
diagnose debug application depmanager 0
diagnose debug disable
diagnose debug reset
```

---

## 18) Atualizar serial number de dispositivo


```bash
execute device replace sn <name> <New_SN>
```

---

## 19) Desabilitar auto-update


```bash
config system admin setting
    set auto-update disable
end
```

## 1) Sessões de administrador


```bash
# Listar sessões de administrador ativas
diagnose system admin-session list

# Encerrar uma sessão específica pelo ID
diagnose system admin-session kill <ID>
```

---

## 2) Debug CLI


```bash
# Debug de comandos CLI executados
diagnose debug cli 8
diagnose debug enable
```

> Útil para ver em tempo real os comandos enviados para dispositivos gerenciados.
