
### 14.1 Fazer backup


```bash
execute backup all-settings [ftp | scp | sftp] <IP:port> <arquivo> <User> <password>
```

### 14.2 Zerar configuração (mantendo IP)


```bash
execute reset all-except-ip
```

### 14.3 Restaurar backup


```bash
execute restore all-settings [ftp | scp | sftp] <IP:port> <arquivo> <User> <password>
```

### 14.5 Migrar entre modelos de FortiManager


```bash
execute migrate all-settings [ftp | scp | sftp] <server> <filepath> <user> <password> {cryptpasswd}
```

### 14.6 Formatar disco

```bash
execute format [disk | disk-ext3 | disk-ext4] <RAID-level> deep-erase <erase-count>
```

### 15.2 FortiManager (restaurar imagem)


```bash
execute restore image ftp <Imagem> <IP_FTP> <User_FTP> <password_FTP>
y
```

## 10) Rollback de configuração

```bash
config system dm
    set rollback-allow-reboot enable
end
```

## 11) Failed Reload


```bash
# Forçar recarga da configuração de um dispositivo
diagnose test deploymanager reloadconf <device>
```

> Útil quando um dispositivo não está aplicando a configuração corretamente após um push.
