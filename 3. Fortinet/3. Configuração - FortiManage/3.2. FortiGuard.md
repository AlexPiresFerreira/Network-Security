### 8.1 O que é distribuído

- Antivírus
- IPS
- Web Filter
- Email Filter

### 8.2 Comandos


```bash
# Listar servidores FortiGuard
diagnose fmupdate view-serverlist fds

# Status de update
diagnose fmupdate update-status fds

# Informações do contrato
diagnose fmupdate dbcontract

# Verificação geral
get fmupdate publicnetwork
get fmupdate service

# Por tipo de serviço
diagnose fmupdate view-serverlist [fds | fct | fgd | fgc | fmtr]
diagnose fmupdate update-status [fds | fct | fgd | fgc | fmtr]

# No FortiGate
diagnose fmupdate dbcontract [<fds | fgd> <SERIAL_NUMBER>]
```
