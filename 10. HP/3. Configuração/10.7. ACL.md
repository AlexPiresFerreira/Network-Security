# ACL

### Bloqueio TELNET interface

``` 
firewall enable
#
acl number 3000 name BLOQUEIO_TELNET
 rule 5 permit ip source 200.231.242.9 0
 rule 10 permit ip destination 200.231.242.9 0
 rule 15 permit ip source 200.230.16.126 0
 rule 20 permit ip destination 200.230.16.126 0
 rule 25 permit ip source 200.255.156.192 0.0.0.63
 rule 30 permit ip destination 200.255.156.192 0.0.0.63
 rule 35 deny tcp destination-port eq telnet
 rule 40 permit ip
#
#
interface Ethernet0/0.37
 firewall packet-filter 3000 inbound
#
```
