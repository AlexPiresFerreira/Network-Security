# DHCP

### DHCP Relay:

```
Sys
#
 dhcp relay server-group 1 ip <IP_Servidor>
 dhcp relay server-group 1 ip <IP_Servidor>
#
#
Interface <interface>
  dhcp select relay
  dhcp relay server-select 1
```

### DHCP na caixa

```
#
dhcp enable
dhcp server forbidden-ip 10.104.72.1 10.104.72.50
#
#
dhcp server ip-pool ECT
gateway-list 10.104.72.1
network 10.104.72.0 mask 255.255.255.0
dns-list 10.0.192.18 10.193.112.40
nbns-list 10.0.192.19 10.192.0.4
domain-name MINUSA.LOCAL
```

#### Verificação
```
display dhcp server ip-in-use all
```
