# NAT

### NAT dinâmico

```
nat address-group 1 <IP_Inicio> <IP_Final>
 
acl number 2020
 rule 0 permit source <REDE+Mask>
 rule 1 permit source <REDE+Mask>
 
Interface <Interface_WAN>
 nat outbound 2020 address-group 1
```
 
### NAT estático com porta 

 ```
Interface <Interface_WAN>
nat server protocol tcp global <IP_WAN> www inside <IP_Inside> www
```

### NAT Reversible

```
#
nat address-group 1 10.251.120.49 10.251.120.49
#
#
acl number 2000
rule 0 permit source 192.168.10.149 0
rule 1 permit source 138.21.138.65 0
#
interface Serial0/0.1 P2P
nat outbound 2000 address-group 1 no-pat reversible
```
