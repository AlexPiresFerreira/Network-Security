# Segurança

### SSH: 

```
local-user <usuario do cliente>
password simple <senha do cliente>
authorization-attribute level 3
service-type ssh
 
user-interface vty 0 4
authentication-mode scheme
protocol inbound ssh
```

### BLOQUEIA TELNET:

```
acl number 3000
rule 0 permit ip source <IP LOOPBACK 0 DO PE> 0
rule 1 permit ip source <IP DO PE> 0
rule 1 permit ip source 200.255.156.192 0.0.0.63

```

### Criação de usuário: 

```
role name rancid-role
 description Rancid role
 rule 1 permit command display *
 rule 2 permit command dir *
 rule 3 permit command undo terminal monitor
 rule 4 permit command screen-length disable
 quit
 
local-user <user> class manage
 password simple <password>
local-user rancid class manage
 service-type telnet
 authorization-attribute user-role rancid-role
```
