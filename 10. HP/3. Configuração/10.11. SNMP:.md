# SNMP

### Habilitar o SNMP

```
snmp-agent
snmp-agent local-engineid 
snmp-agent sys-info version all (Versão)
 
- Determinar a community e modo leitura ou escrita
snmp-agent community <read | > <community> {acl <acl>}
 
- Determinar o servidor SNMP
snmp-agent target-host trap address udp-domain <IP_Servidor> params securityname <community>
```

### SNMPv3

```
snmp-agent group v3 saga authentication
snmp-agent usm-user v3 rosagauser saga authentication-mode md5 saga@2016 privacy-mode des saga@2016
snmp-agent target-host trap address udp-domain 192.6.1.233 params securityname rosagauser v3
```
